# Quick Start Guide: Advanced Resume Builder

## üöÄ Getting Started

### Step 1: Open the Advanced Builder
1. Go to the **Resume Builder** page
2. Click the **"Advanced Builder"** button (blue button with layout icon)
3. A modal will open with the draggable resume builder

### Step 2: Fill in Your Information

#### Personal Information (Required)
- **Full Name**: Your complete name
- **Email**: Professional email address
- **Phone**: Contact number
- **Location**: City, State/Country
- **Title**: Your professional title (e.g., "Software Engineer")

#### Professional Summary
- Write a brief 2-3 sentence summary of your professional background
- Add your career objective if desired

#### Skills & Interests
- **Technical Skills**: Type a skill and press Enter or click Add
  - Example: JavaScript, Python, React, AWS
- **Soft Skills**: Leadership, Communication, Problem-solving
- **Interests**: Open Source, AI/ML, Photography
- Click the √ó on any tag to remove it

### Step 3: Add Work Experience
1. Click **"Add Experience"** button
2. Fill in:
   - Company name
   - Position/title
   - Location
   - Dates (or check "Currently working here")
   - Responsibilities (use bullet points)
   - Key achievements
3. Add more experiences as needed
4. Click trash icon to remove any entry

### Step 4: Add Education
1. Click **"Add Education"** button
2. Enter:
   - Institution name
   - Degree (e.g., Bachelor of Science)
   - Field of study (e.g., Computer Science)
   - Location
   - Dates
   - GPA (optional)
   - Achievements (optional)

### Step 5: Add Projects
1. Click **"Add Project"** button
2. Include:
   - Project name
   - Your role
   - Technologies used
   - Start and end dates
   - Description
   - Key highlights (achievements, metrics, impact)
   - Project URL (if available)

### Step 6: Add Certificates
1. Click **"Add Certificate"** button
2. Fill in:
   - Certificate name
   - Issuing organization
   - Issue date
   - Expiry date (if applicable)
   - Credential ID
   - Verification URL

### Step 7: Optional Sections

#### Volunteering
- Organization name
- Your role
- Dates
- Description of work
- Impact made

#### Awards & Achievements
- Award title
- Who issued it
- Date received
- Description

#### Languages
- Language name
- Proficiency level (dropdown: Native, Fluent, Advanced, Intermediate, Beginner)
- Certification (e.g., TOEFL 110, IELTS 8.5)

#### Publications
- Publication title
- Authors (comma-separated)
- Publisher
- Publication date
- URL
- Description

#### Website & Social Links
- Personal website
- LinkedIn profile URL
- GitHub profile URL
- Portfolio URL
- Twitter/X profile

## üí° Pro Tips

### Writing Tips
- **Use action verbs**: Led, Developed, Implemented, Achieved, Optimized
- **Include metrics**: "Increased performance by 40%", "Managed team of 5"
- **Be specific**: Instead of "Worked on project", say "Developed React-based dashboard serving 10K+ users"
- **Keep it concise**: Bullet points over paragraphs
- **Quantify achievements**: Numbers stand out

### Organization Tips
- **Put most important sections first**: Usually Experience, then Education
- **Recent items first**: Most recent job/degree at the top
- **Relevant content**: Tailor to the job you're applying for
- **One page if possible**: Especially for early career

### Skills Section Tips
- **Group by category**: Frontend, Backend, DevOps, etc.
- **Proficiency level**: Consider adding (Advanced), (Intermediate) tags
- **Relevant only**: Include skills relevant to your target role
- **Trending technologies**: Include modern, in-demand skills

### Technical Details
- **Consistent date format**: Pick MM/YYYY or Month Year and stick with it
- **No pronouns**: Avoid "I", "me", "my" in descriptions
- **Parallel structure**: Start similar items the same way
- **Proofread**: Check for typos and grammar

## ‚ùì Common Questions

**Q: Can I save a draft and come back later?**
A: Not currently. Complete your resume in one session or copy your data before closing.

**Q: Can I edit my resume after saving?**
A: You'll need to create a new version. The builder doesn't currently load existing resumes for editing.

**Q: How many items can I add to each section?**
A: There's no limit! Add as many projects, jobs, certificates as you need.

**Q: Can I customize the order of sections?**
A: Yes! Drag and drop sections to reorder them.

**Q: Will my resume be analyzed by AI?**
A: Yes! After saving, you can run AI analysis just like uploaded resumes.

**Q: Can I download as PDF?**
A: The builder saves as text. You can then use other tools to convert to PDF, or use the Quick Build option which has built-in PDF download.

**Q: What happens if I close the modal?**
A: Your data will be lost unless you clicked "Save Resume". Make sure to save before closing!

**Q: Can I have multiple resumes?**
A: Yes! Create different versions for different jobs by building multiple resumes.
